# tonmai-toolkit #
Support Maya 2022 - 2025
Created by Natchapon Srisuk

# Installation #
1. Copy "tonmaiToolkit.py" in plug-ins folder to your maya plug-ins folder
2. Copy "TonmaiToolkit" FOlder in scripts folder to your maya scripts folder
3. Open Maya and Click Load plugin tonmaiToolkit.py in plugin editor.
